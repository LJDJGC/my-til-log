### 💡 学びのポイント
**新規習得：**
- **localStorageへの保存と読み込み**: ブラウザの記憶領域にデータを保存し、リロードしてもデータを保持する技術を学んだ。
- **JSON形式への変換**: 配列やオブジェクトなどの複雑なデータを、文字列として保存するための `JSON.stringify` と、復元するための `JSON.parse` の役割を理解した。

**理解深化：**
- **useEffectの Dependency Array**: 第2引数に空配列 `[]` を渡すと「マウント時（最初）に1回だけ」実行され、変数を入れると「その変数が変化するたびに」実行されるという挙動を、実体験を 통해習得した。
- **副作用（Side Effect）の概念**: 関数型プログラミングにおける「関数は入力に対して出力を返すだけ」という原則に対し、データの取得や保存など「他への影響を与える処理」を副作用と呼び、Reactではこれを useEffect で扱うことを知った。

    if (savedSessions) {
      setSessions(JSON.parse(savedSessions));
    }
  }, []);

  // 保存処理（sessionsが変わるたびに実行）
  useEffect(() => {
    localStorage.setItem('sessions', JSON.stringify(sessions));
  }, [sessions]);
