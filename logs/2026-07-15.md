### 💡 学びのポイント
**新規習得：**
- **JSX グリッドレイアウトの実装パターン**: カレンダーを「月ラベル行」＋「曜日行×週列」の2層構造で表現。`monthLabels.map()` でラベル行、`["Mon",...,"Sun"].map()` で各行、その内側で `weekList.map()` で列（週）を展開する。
  ```tsx
  {/* 月ラベル */}
  <div>
    {monthLabels.map((label, i) => (
      <span key={i}>{label ?? ""}</span>
    ))}
  </div>
  {/* 曜日行 x 週セル */}
  {["Mon", "Tue", ...].map((dayName, dayIndex) => (
    <div key={dayIndex}>
      <span>{dayName}</span>
      {weekList.map((week, weekIndex) => {
        const dateStr = week[dayIndex];
        const dayData = dayMap.get(dateStr);
        const level = getLevel(dayData, "study");
        const colorClass = TYPE_COLORS["study"][level];
        return <span key={weekIndex} className={colorClass}>■</span>;
      })}
    </div>
  ))}
  ```
- **型アノテーションの重要性**: TypeScript は関数の引数から型推論できない場合、`any`（暗黙の any）になる。`getLevel(dayData, type)` のように外部から呼ばれる関数では、明示的な型注釈が必要。
