- `return` a function defined within an IIFE (Immediately Invoked Function) as an object, allowing only a specific function (`playRound`) to be executed externally.

**Deeper Understanding:**
- Instead of writing all conditional branching within the main process (`playRound`), extracting "single-responsibility functions that only perform determination," such as `checkWinner` and `checkDraw`, dramatically improves code readability.
