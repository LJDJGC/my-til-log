### 💡 学びのポイント
**新規習得：**
- **`dateKey` 関数のフォーマットパターン**: Date型 → "YYYY-MM-DD" 文字列への変換。`String(d.getMonth() + 1).padStart(2, "0")` のイディオム。月は0始まり（0=1月）なので +1 が必要。
  ```tsx
  function dateKey(d: Date): string {
      return `${d.getFullYear()}-${String(d.getMonth() + 1).padStart(2, "0")}-${String(d.getDate()).padStart(2, "0")}`;
  }
  ```
- **`aggregateByData`（※typo: 正しくは aggregateByDate）の2パス集計**: 1パス目で日付ごとの合計時間とタイプ別内訳を蓄積、2パス目で `Object.entries()` を回して最も時間の長いタイプを `dominantType` に決定する設計。`map.values()` でValueのイテレータを直接ループできる。

- **`useMemo` チェーンの依存関係グラフ**:
  ```
  sessions → { dayMap, maxDayMinutes }  ← sessions依存
  weeks    → days[]                      ← weeks依存
  days[]   → weeksList[][]               ← days依存
  weeksList → monthLabels[]              ← weeksList依存
  ```
  各useMemoの戻り値が次のuseMemoの入力になる直列依存。Reactはこの依存関係を追跡して、変更があった処理だけを再実行する。

- **`getLevel` の5段階強度判定ロジック**: 0〜4のレベルを、最大値に対する比率の非線形閾値で決定。
  - ratio = 0 → level 0（活動なし）
  - ratio <= 0.25 → level 1（低）
  - ratio <= 0.5 → level 2（中）
  - ratio <= 0.75 → level 3（高）
  - それ以上 → level 4（最大）
  0.1以下を level 0 に分類する最初のチェックがユニーク（完全にゼロでなくても「実質ゼロ」扱い）。

- **`dateKey` Format Idiom**: The TypeScript/JavaScript standard pattern for converting Date to "YYYY-MM-DD" — `String(d.getMonth() + 1).padStart(2, "0")`. Month is 0-indexed (0 = January) so `+1` is required.
