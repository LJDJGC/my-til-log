### 💡 学びのポイント
**新規習得：**
- **ノンヌルアサーション `!` の正確な意味**: `map.get(key)!` は「型上は undefined の可能性があるけど、人間がここでは絶対に存在することを保証する」という TypeScript への表明。論理的には存在が確定していても、TS の型推論の限界で追えないケースで使う。
- **`|| 0` ガードパターン**: `day.byType[session.type]` が初アクセス時は `undefined` になる。`undefined += duration` は `NaN` になるので、`(day.byType[session.type] || 0)` で安全に加算する。これは TypeScript/JavaScript の定石パターン。

**理解深化：**
- **関数をコンポーネントの外に出す理由**: コンポーネント内に関数を定義すると、再レンダリングのたびに新しい関数オブジェクトが生成される。処理自体が sessions の変更にしか依存しない純粋関数なら、外に出して `useMemo` でラップするのが正しい設計。
- **復習の効果実感**: 「読める・わかる」と「書ける」の差を実感。1回目の実装時に見落としていた `|| 0` のようなエッジケースに、自分で書こうとして初めて気づけた。

```tsx
function aggregateByDate(sessions: Session[]): Map<string, DayData> {
  const map = new Map<string, DayData>();
  for (const session of sessions) {
    if (!map.has(session.date)) {
      map.set(session.date, {
        date: session.date,
        total: 0,
        byType: {},
        dominantType: "",
      });
    }
    const day = map.get(session.date)!;
    day.total += session.duration;
    day.byType[session.type] = (day.byType[session.type] || 0) + session.duration;
  }
  return map;
}
```

### 💡 Key Learnings
**Newly Acquired:**
- **Non-null Assertion `!`**: `map.get(key)!` tells TypeScript "I guarantee this value exists even though the type says it could be undefined." Used when logical flow guarantees existence but TS's type inference can't track it.
- **`|| 0` Guard Pattern**: `day.byType[session.type]` is `undefined` on first access. `undefined += duration` produces `NaN`. The `(value || 0)` pattern is the idiomatic JavaScript/TypeScript way to safely accumulate.

**Deeper Understanding:**
- **Why Move Functions Outside Components**: Defining a function inside a component creates a new function object on every render. If it's a pure function that only depends on props, define it outside and wrap with `useMemo`.
- **Recall Effect in Practice**: The gap between "can read/understand" and "can write" is real. Edge cases like `|| 0` that were overlooked during the first implementation only surface when writing from scratch.
