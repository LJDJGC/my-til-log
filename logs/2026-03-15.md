### 💡 学びのポイント
**新規習得：**
- **クラスの基本構文:** ES6から導入された `class` キーワードを使った構文。クラスは内部的には「特別な関数」であること。
- **constructor（コンストラクター）:** インスタンス生成時（`new` を呼んだとき）に自動的に実行され、オブジェクトの初期化を行う特別なメソッド。

**理解深化：**
- **Factory Functionsとの違い:** 現在「Tic Tac Toe」で使っているファクトリー関数やモジュールパターンと、今回学んだ `class` 構文が、どちらも「オブジェクトを生成・管理するアプローチ」であるという点の繋がり。

**トラブルシューティング：**
- 特になし（インプット中心のため）

### 🔗 実装コード
- リポジトリ：未作成（学習メモ・検証フェーズ）
- 主要コード：
```javascript
// クラス宣言とコンストラクターの基礎検証
class Player {
  // インスタンス化されるときに呼ばれる初期化処理
  constructor(name, marker) {
    this.name = name;
    this.marker = marker;
  }

  // クラス内のメソッド
  introduce() {
    console.log(`I am ${this.name} and my marker is ${this.marker}`);
  }
}

const player1 = new Player('Alice', 'X');
player1.introduce(); // "I am Alice and my marker is X"
```
