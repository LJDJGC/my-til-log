### 💡 学びのポイント
**新規習得：**
- **「前週の月を覚えておく」パターン**: 配列をループしながら「前回の値」を保持するには、ループの外で変数を宣言し、ループ内で更新する。
  ```tsx
  let prevMonth = "";
  weeksList.map((week) => {
    const currentMonth = formatMonthLabel(week[0]);
    if (currentMonth !== prevMonth) {
      prevMonth = currentMonth;
      return currentMonth;
    }
    return null;
  });
  ```
