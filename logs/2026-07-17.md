### 💡 学びのポイント
**新規習得：**
- **APIキーの「アプリケーション制限」とサーバー間通信の関係**:
  Route Handler (サーバー) → YouTube API (外部サーバー) の通信には Referer ヘッダーが付かない。
  そのため「HTTPリファラー制限」を設定すると、サーバーサイドからのリクエストはすべてブロックされる。
  解決策：API制限で使用できるAPIを限定することで、キーが漏れても被害を最小限にする。
- **Route Handler の `export async function GET`**:
  Next.js App Router では、`src/app/api/xxx/route.ts` に `export async function GET()` と書くだけで
  `GET /api/xxx` のエンドポイントになる。関数名がHTTPメソッドに対応。
- **`response.ok` の使い方**:
  `fetch()` のレスポンスには `ok` プロパティがあり、HTTPステータスが200番台なら `true`、それ以外なら `false`。
  エラーハンドリングで便利。`if (!response.ok)` で「失敗した場合」を表現できる。

**理解深化：**
- **Route Handler は「中継役（プロキシ）」**:
  ブラウザ → 自サーバー → YouTube のように、間に入ることでAPIキーを隠せる。
  ブラウザから見えるのは自サーバーのURLだけ。キーは最後までブラウザに渡らない。
- **`?.`（オプショナルチェイニング）と `||` の使い分け**:
  ```tsx
  // ?. → プロパティが存在したらアクセス、なければ undefined
  // || → 左が falsy（undefined, null, 空文字, 0）なら右を使う
  item.snippet.thumbnails?.medium?.url || item.snippet.thumbnails?.default?.url || ""
  ```
  `?.` で安全に深いプロパティにアクセスし、`||` でフォールバック値を設定する。
- **`.map()` で「変換」する**:
  APIの生レスポンスをアプリ用のシンプルな形に変換する。元のデータを変更するのではなく、
  必要な項目だけを抜き出した「新しい配列」を作る。

```tsx
export async function GET(request: NextRequest) {
  const { searchParams } = new URL(request.url);
  const query = searchParams.get("q");
  if (!query || query.trim().length === 0) {
    return NextResponse.json({ error: "required" }, { status: 400 });
  }
  const apiKey = process.env.YOUTUBE_API_KEY;
  const url = new URL("https://www.googleapis.com/youtube/v3/search");
  url.searchParams.set("part", "snippet");
  url.searchParams.set("q", query);
  url.searchParams.set("key", apiKey);
  url.searchParams.set("maxResults", "10");
  url.searchParams.set("type", "video");
  const response = await fetch(url.toString());
  const data = await response.json();
  const results = (data.items || []).map((item) => ({
    videoId: item.id.videoId,
    title: item.snippet.title,
    channelTitle: item.snippet.channelTitle,
    thumbnailUrl: item.snippet.thumbnails?.medium?.url || "",
  }));
  return NextResponse.json({ results });
}
```

- **`.map()` for transformation**: Creates a new array with only the needed fields.
